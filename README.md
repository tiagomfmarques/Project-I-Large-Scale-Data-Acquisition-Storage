# **Sistema de Aquisição e Pesquisa de Artigos Científicos e Datasets**

---
[![Python 3.10](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)](https://www.python.org/downloads/)
[![Redis 7](https://img.shields.io/badge/Redis-DC382D?style=for-the-badge&logo=redis&logoColor=white)](https://redis.io/downloads/)
[![Docker](https://img.shields.io/badge/docker-257bd6?style=for-the-badge&logo=docker&logoColor=white)](https://www.docker.com/get-started)


Este projeto foi desenvolvido para a unidade curricular de Ciência de Dados em Larga Escala. O sistema automatiza a recolha de artigos científicos a partir das plataformas Crossref e OpenAlex, inclui também a realização de web scraping de datasets da UCIrvine, e permite a sua indexação e pesquisa eficiente em larga escala utilizando Redis e Docker.
