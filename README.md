# **Sistema de Aquisição e Pesquisa de Artigos Científicos e Datasets**

---
[![Python 3.10](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![Elasticsearch 8.10](https://img.shields.io/badge/elasticsearch-8.10+-blue.svg)](https://www.elastic.co/downloads/elasticsearch)
[![Docker](https://img.shields.io/badge/docker-20.10+-blue.svg)](https://www.docker.com/get-started)


Este projeto foi desenvolvido para a unidade curricular de Ciência de Dados em Larga Escala. O sistema automatiza a recolha de artigos científicos a partir das plataformas Crossref e OpenAlex, inclui também a realização de web scraping de datasets da UCIrvine, e permite a sua indexação e pesquisa eficiente em larga escala utilizando Elasticsearch e Docker.
